### T-WELCOME-2: Splash-карточка на входе ✅
**Приоритет:** 🔴 | **Статус:** Выполнено (commit 97b0429)

**Цель:** Компактная карточка при первом входе. Один раз за жизнь пользователя.

**Дизайн:** 4 плитки с цифрами (1000+ разборов, 20 эфиров, 9 гайдов, 99 добавок), заголовок «Твой навигатор по здоровью». Модальный оверлей. Nunito, тёплые цвета (#faf8f4, #f5f0e8), терракотовый акцент (#c4724a), border-radius 14px.

**Кнопки:**
- «Подробнее» → `welcome_seen='details'` в localStorage, трекнуть `welcome_splash_details`, открыть тур (T-WELCOME-3)
- «Пропустить» → `welcome_seen='splash_skipped'`, трекнуть `welcome_splash_skipped`, закрыть

**Логика:** `localStorage.welcome_seen` не существует → показать. Любое значение → не показывать. Показывать ВСЕМ включая deep link. Трекнуть `welcome_splash_shown` при показе.

**Проверка:** dev в инкогнито → splash → «Пропустить» → больше не появляется.

### T-WELCOME-3: Welcome-тур (6 экранов) ✅
**Приоритет:** 🔴 | **Статус:** Выполнено (commit 97b0429)

**Цель:** 6 экранов-слайдов с реальным меню в плиточном формате. Открывается из splash по «Подробнее».

**Полная спецификация:** файл `onboarding-spec.md` в проекте Claude.ai. Ключевые моменты:

**Сетка:** `grid-template-columns: repeat(2, 1fr); gap: 8px`. Карточки: белый фон, border-radius 14px, иконка 48x48 на цветном фоне, название 12px bold. Бейджи NEW/FREE/PRO/ХИТ/-20%.

**6 экранов (тултип + подсветка 1-2 карточек, остальные opacity 0.2):**
1. Полезные игры (фиолетовый, #f0eaff)
2. Подборки + Бесплатные гайды (оранжевый, #fff8f0)
3. Эфиры + Подкасты (фиолетовый, #f0f0ff) + автоскролл + «↓ листай вниз»
4. Добавки + Скидка на анализы (зелёный, #edf8f3)
5. Гайды Pro + Консультация (розовый, #fdf0f4)
6. Мои беспокойства → квиз (золотой, #fff5e6)

**Порядок плиток в сетке:**
```
[Полезные игры NEW]      [Мои беспокойства]
[Подборки по темам]      [Полезные материалы FREE]
[Гайды Pro PRO]          [Консультация]
[Волшебные таблетки ХИТ] [Скидка 20% на анализы -20%]
[Отзывы]                 [Эфиры]
[Подкасты]
```

**Навигация:** «Дальше»/«Назад», touch swipe, «✕» крестик, «Пропустить», 6 точек-индикаторов (активная #c4724a). Экран 6: «Подобрать для меня →» → квиз.

**Автоскролл (экран 3):** scrollTop=0, через 200мс scrollIntoView к подсвеченной карточке.

**Флаги:** закрыл → `welcome_seen='tour_closed'`, дошёл → `welcome_seen='tour_completed'`, «Подобрать» → квиз. Каждый переход → трек `welcome_tour_step` (step_number).

**Проверка:**
1. Инкогнито → splash → «Подробнее» → тур 6 экранов
2. Экран 3 — автоскролл к Эфирам/Подкастам
3. «✕» → закрыл → больше не появляется
4. Экран 6 → «Подобрать для меня» → квиз
5. После квиза → промпты не показываются
6. После закрытия тура без квиза → промпты работают
